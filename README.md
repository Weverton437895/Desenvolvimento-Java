# Desenvolvimento Java 

Repositório com meus estudos e projetos desenvolvidos em Java ao longo da graduação.

Este repositório demonstra minha evolução desde os fundamentos de Programação Orientada a Objetos até o desenvolvimento de aplicações web com Spring Boot.

---

## Estrutura do Repositório

### tecnicas-programacao-1
Estudos de Java focados em fundamentos e Programação Orientada a Objetos (POO).

Conteúdos abordados:
- Estruturas condicionais e de repetição
- Métodos
- Classes e Objetos
- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Interfaces
- Manipulação de arquivos

Também foram desenvolvidas aplicações com interface gráfica utilizando **Java Swing**, trabalhando com:
- JFrame
- JButton
- JTextField
- Eventos (ActionListener)
- Organização de layouts

---

### tecnicas-programacao-2
Foco em desenvolvimento backend com Spring Boot.

Conteúdos abordados:
- Criação de APIs REST
- Controllers
- Services
- Repositories
- Integração com banco de dados
- Arquitetura em camadas
- Injeção de dependência

---

## Tecnologias Utilizadas

- Java
- Java Swing
- Spring Boot
- Maven / Gradle
- Programação Orientada a Objetos
- API REST

---

## Objetivo

Consolidar meus conhecimentos em desenvolvimento backend utilizando Java e Spring Boot, aplicando boas práticas de organização e arquitetura.
